# Roadmap

## Near Term

- real Wayland layer-shell backend
- provider invalidation and event handling hardening
- richer formatter capabilities
- config reload workflow
- stronger smoke and CI coverage

## Mid Term

- external provider extension ABI or process bridge
- better visual styling, icons, and text shaping
- multi-output/surface policy
- tighter observability and profiling hooks

## Later

- optional shared Lua/config library extraction for `zbar`, `zide`, and
  `wayspot`
- deeper sibling integration once the contracts are stable
